# AMD District Map Prototypes

Experimental map-module directions for the Avian Municipal District hub.

These files are review artifacts only. Do not treat them as production replacements for `../index.html` unless one is explicitly promoted.

## Files

- `option-4-property-gis-2_5d.html`  
  Current direction: a property-record GIS interface with a synthetic 2.5D building layer and nest record symbols.

- `site-preview-option-4-property-gis-2_5d.html`  
  Generated local preview of the AMD hub with only the map section swapped for option 4.

- `screenshots/`  
  Rendered review screenshots.

## Design Rule

Target a 99% familiar civic/property map and 1% Avian Municipal District displacement. The map should read like a normal municipal web GIS or property-record lookup first. The bird-world difference should live in small institutional details: record language, nest symbols, Clerk attribution, and units.
